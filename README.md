# Video-App
### This is the Video-App api repository.

## Prerequisites:
Node.js >= 8.9.x/NPM

## Setup:
* clone this repository

    ```
    $ git clone https://github.com/djgovani/Video-app-express-api.git
    ```

* Run below command to install required packages.

    ```
    npm install
    ```

* For development

    ```
    npm run dev
    ```

* For production

    ```
    npm start
    ```
